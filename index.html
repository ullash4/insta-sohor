How does javascript Works ?

HTML and CSS build websites structure and javascript give website life to run. Javascript is a client-side scripting language. But javascript needs to be enabled on the client's browser. That's why most browser supports javascript and they have their own javascript engine. Such chrome has V8, firefox has Spider monkey, safari has javascript core, and edge has chakra. W`e have used a chrome browser to run our program that has the V8 JavaScript engine, which is also used for creating Node.js. JavaScript is an interpreted language that means it gets executed in line by line manner. This means the JavaScript engine converts the Js code line by line and runs in the same manner instead of converting the whole program once.
